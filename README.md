# RthGPT
A dumb version of GPT2, based on the paper and the explainations I could find on the net.

## Why
The goal of this project is to understand more deeply how Generative Pretrained Transformers work inside.
Also perhaps make a cool custom chatbot for me and friends.

## How
A lot of tensorflow, and a thousand edits you don't see because I don't commit often.
Since *Big Machines* cost a lot of money to buy, and even rent, this model will be relatively small.

TLDR; making a small GPT for learn & fun.*
